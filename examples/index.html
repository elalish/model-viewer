<model-viewer> - Smoke Tests

There are several HTML pages containing usages of the <model-viewer> element with different configurations for testing features.

Tests

  • core.html: Tests using defaults and attributes not covered by other tests.
  • sources.html: Tests using different 3D formats, <source> elements, and error scenarios loading a 3D file.
  • ar.html: Tests using the ar attribute.
  • poster.html: Tests using the poster attribute.
  • preload.html: Tests using the preload attribute.
